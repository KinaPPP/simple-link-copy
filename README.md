# シンプルリンクコピー / Simple Link Copy

ページタイトルとURLを1クリックでコピーできるChrome拡張機能です。  
Twitter / Bluesky などへの投稿リンクを素早く作るために設計されました。

![Simple Link Copy](icon128.png)

---

## 機能

- **タイトル＋URLをコピー**  
  現在のタブのページタイトルとURLを改行区切りでコピーします。

- **テキスト選択モード**  
  ページ上でテキストを選択した状態でポップアップを開くと、  
  選択テキスト＋URLをコピーします。

- **サイト名を省略**  
  `« サイト名` `| サイト名` `：サイト名` `｜サイト名` などの末尾パターンを自動検出して除去できます。  
  タイトル内の `[MODE: ...]` のような括弧内のセパレータは誤検出しません。  
  設定はON/OFFを記憶します。

- **Twitter文字数カウント（全角換算140字ベース）**  
  全角=1字、半角=0.5字、URL=11.5字換算でリアルタイム表示します。

- **コピー後に自動で閉じる**  
  コピー完了後800ms後にポップアップが自動的に閉じます。

## インストール方法

Chrome Web Storeには未公開です。手動インストールが必要です。

1. このリポジトリをZIPでダウンロードして展開する
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. 展開したフォルダを選択する

---

## プライバシー

本拡張機能が取得する情報は以下のみです：

- 現在のタブのページタイトル
- 現在のタブのURL
- ページ上で選択されたテキスト（選択時のみ）

**外部サーバーへのデータ送信は一切行いません。**
